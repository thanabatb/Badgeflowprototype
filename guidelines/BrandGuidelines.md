# BadgeFlow Brand Guidelines

## Brand Palette

| Role | Name | Hex | Primary Usage |
| --- | --- | --- | --- |
| Primary | Flow Blue | `#4F6DF5` | CTAs, logo, headings, links |
| Primary Dark | Deep Ink | `#1E2A4A` | Body text, dark surfaces |
| Accent | Spark Coral | `#FF6B6B` | Notifications, highlights, hover accents |
| Secondary | Mint Fresh | `#34D399` | Success states, checkmarks, earned indicators |
| Neutral Light | Cloud | `#F5F7FA` | Page backgrounds, card fills |
| Neutral Mid | Slate | `#94A3B8` | Captions, secondary text, borders |
| Pop | Sunny Gold | `#FBBF24` | Stars, premium badge accents only |

## Usage Rules

- Flow Blue is the hero. Use it for around 60% of brand-colored elements.
- Spark Coral is the attention-grabber. Never exceed 15% of a composition.
- Use Deep Ink for all body text. Never use pure black `#000000`.
- Cloud is the default background. Prefer Cloud over pure white.
- Sunny Gold is seasoning, not a main course. Use it for stars/badges only.
- Flow Blue + Cloud is the safest default pairing.
- Never place Coral text on Blue backgrounds (accessibility failure).

## UI Mapping (Quick Reference)

- Primary button: `Flow Blue` background + white text, Coral on hover.
- Secondary/outline button: Cloud/white surface + Slate border + Deep Ink text.
- Success chips/toasts/check states: Mint Fresh.
- Error/warning emphasis: Coral (with readable contrast on light surfaces).
- Default canvas/card background: Cloud.

## Accessibility Guardrails

- Keep normal body text contrast at least WCAG AA (4.5:1).
- Use Deep Ink for content-heavy paragraphs and long-form reading.
- Avoid saturated color-on-color combinations unless contrast is verified.
- Treat Coral as accent only, not as base text color for long passages.
